Objetos JavaScript

Um objeto JavaScript é uma coleção de propriedades, e uma propriedade é uma associação entre um nome (ou chave) e um valor. O valor de uma propriedade pode ser uma função, que é então considerada um métrodo do objeto.

Os objetos JvaScript são usados para armazenar dados e funcionalidades relacionadas. Eles podem ser criados usando a sintaxe de objeto literal, a função construtora ou a classe.

Exemplo de Objeto Literal:

const pessoa = {
        nome: "João";
        idade: 30;
        saudacao: function() {
            console.log("Olá, meu nome é" + this.nome);
    }
}
